# Planning Checklist

- [ ] Goal is stated in one sentence.
- [ ] Success criteria are explicit.
- [ ] Scope exclusions are listed.
- [ ] Relevant files or modules are identified.
- [ ] Run directory and worktree paths are listed.
- [ ] Builder responsibilities are listed.
- [ ] Verifier responsibilities are listed.
- [ ] Subagents are justified or explicitly unnecessary.
- [ ] Risks are named before implementation.
- [ ] Human Owner has received the planning report.
